# Week 2 — Advanced Prompting & AI Tools

## Overview

Week 2 moves beyond prompting fundamentals in two directions — deeper technique and practical application. The first document covers structured frameworks for complex reasoning, perspective-aware prompting, and iterative refinement. The second covers the tools that turn prompting into real workflows: research assistants, data analysis, local models, and personalised AI workspaces.

## Key Concepts Covered

**Advanced Prompt Engineering**
- CO-STAR framework for multi-step complex prompting
- Least to Most Prompting for sequential problem decomposition
- SimTom Prompting for perspective-aware reasoning
- Self-Refine, Self-Verification, and Chain of Verification
- Prompt creation workflows using multiple models

**From Prompting to Prototyping**
- Google NotebookLM as a personalised research assistant
- Retrieval-Augmented Generation (RAG) — how grounded AI works
- Google Colab for natural language data analysis
- LM Studio for running AI locally on private data
- Perplexity Spaces for persistent, domain-specific AI assistants
