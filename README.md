# AI Learning

A structured, week-by-week documentation of my AI learning journey — covering theory, applied techniques, tools, and hands-on practice. Each week focuses on a specific domain, building progressively from fundamentals to advanced applications.

---

## Why This Exists

Most people learn AI by jumping straight into tools. This repository takes a different approach — understanding how things work before using them. The goal is to build genuine depth, not just surface familiarity.

Each week produces two things: documented understanding of the concepts covered, and applied work that puts those concepts into practice.
